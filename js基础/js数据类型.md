**基本类型:** number, null, undefined, string, boolean, symbol

**引用类型**: object



**Symbol**:  es6引入,表示独一无二的值

基本类型存放在栈(stack)里,按值访问

基本类型值不可变

基本类型的比较是他们值的比较

引用类型Object细分有: Object,Array, Function, Date, RegExp等

引用类型按引用访问

引用类型值是可变的

引用类型比较是引用的比较

引用类型保存在堆中,栈中只保存一个引用变量,类似指针

**三个包装类型**

String, Number, Boolean
